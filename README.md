# 🧪 GlassLab — Glassmorphism Design Playground

**GlassLab** is an interactive, real-time Glassmorphic UI generator built for modern front-end developers and designers. Easily design stunning glassmorphism components with granular control and export production-ready **CSS** or **Tailwind CSS** code with a single click.

---

## ✨ Features

*   **Live Preview:** Adjust colors, opacity, blur, shadow, and corner radius, and see your design render instantly on the interactive central card.
*   **Curated Presets:** High-quality presets ready to use, including:
    *   *Classic Glass*
    *   *Apple VisionOS* (Aerodynamic glassmorphism with custom styling)
    *   *Windows Acrylic* (Fluent Design concept)
    *   *Soft Glass, Dark Card, Minimal*, and more.
*   **Dual-Format Export:** Instantly copy either clean, optimized standard **CSS** or dynamic **Tailwind CSS** classes.
*   **Saved Designs (Favorites):** Save your favorite combinations using the **⭐ Favorite** button. Saved components persist locally in your browser via `LocalStorage` and can be re-applied instantly with the **⚡ Apply** action.
*   **Responsive Layout:** Fully optimized for all viewports, ensuring a seamless experience across desktop, tablet, and mobile displays.

---

## 🛠️ Tech Stack

This project is built using modern web standards without heavy frameworks or unnecessary third-party dependencies:

*   **HTML5** & **CSS3** (Using modern Custom Properties / CSS Variables and Flexbox/Grid layouts)
*   **JavaScript (ES6+)** — Structured as modular ES modules (`import/export` architecture)
*   **Local Storage API** — For persistent, client-side storage of favorite styles

---

## 🚀 Quick Start

No installations (`npm install`) or local build configurations are required. Simply run:

1.  Clone the repository:
    ```bash
    git clone [https://github.com/your-username/glasslab.git](https://github.com/your-username/glasslab.git)
    ```
2.  Navigate to the project folder:
    ```bash
    cd glasslab
    ```
3.  Open `index.html` in your browser (or run it via VS Code's **Live Server** extension).
